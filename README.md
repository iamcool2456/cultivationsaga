Quick hosting (GitHub Pages):

1) Push this repo to GitHub on the `main` branch.
2) In GitHub: **Settings → Pages → Build and deployment → GitHub Actions**.
3) Wait for the **Deploy to GitHub Pages** workflow to finish.

Your live site will be:
`https://<your-username>.github.io/<your-repo-name>/`

See `.internal/README.md` for local dev, QA, and other deployment options.
